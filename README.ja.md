# 🎵 ミュージックプレーヤー

> ビンテージレコード風インターフェースを備えた、純粋なHTML製ミュージックプレーヤー。150以上の言語と約50の地域方言に対応しています。**たった1つのHTMLページで、世界中の方言のスローメロディーをお楽しみください。**

**注：このプロジェクトは、人工知能の支援を受けて作成されました。**

---

## ✨ 特長

- 🎨 **没入型レコードデザイン** – 回転するディスク、トーンアームのアニメーション、レトロなビジュアル体験。
- 🌍 **150以上の言語＆50の方言** – 呉語、広東語、客家語、贛語、湘語、晋語などを含む。
- 📂 **ドラッグ＆ドロップ対応** – オーディオファイル（MP3/WAV/FLAC/OGG）をドラッグするか、「音楽を追加」をクリックするだけ。
- 📝 **LRC歌詞同期** – `.lrc`ファイルをインポートして、ハイライト表示のスクロール歌詞とフルスクリーン歌詞モードに対応。
- 🎛️ **フル再生コントロール** – 再生/一時停止、前へ/次へ、シャッフル、リピートモード（なし/全曲/1曲）およびボリューム調節。
- 📱 **レスポンシブ対応** – デスクトップ、タブレット、スマートフォンで快適に動作。
- 🧩 **外部依存ゼロ** – ブラウザだけで動作し、インストール不要。

---

## 🚀 クイックスタート

1. `TheMusicPlayer.html` ファイルをダウンロードします。
2. 任意の最新ブラウザ（Chrome、Edge、Firefoxなど）で**ダブルクリック**して開きます。
3. **「音楽を追加」** をクリックするか、オーディオファイルをウィンドウにドラッグ＆ドロップします。
4. **「歌詞」** ボタンをクリックすると、現在の曲に `.lrc` 歌詞ファイルをインポートできます。
5. **歯車アイコン（設定）** をクリックすると、フォントやインターフェース言語を切り替えられます。

### 詳細

このミュージックプレーヤーは、ユーザーが喧騒から離れ、美しいメロディーとともに穏やかなスローライフを楽しめるように作られています。150の世界言語と約50のユニークな地域方言をカバーする包括的なローカライズを備えています。初めて、没入型のヴィンテージレコードビジュアルデザインを再生インターフェースに統合し、より質感とノスタルジックなリスニング体験を実現しました。ぜひダウンロードしてお試しください。

> **デザインの意図：** 私の核心的なインスピレーションは、リラックスしたシンプルな方法で音楽を楽しみたいという単純な願いから生まれました。このプレーヤーを純粋なHTML技術で構築したのは、すべてのパソコンでユニバーサルな互換性を実現できるからです。これにより、すべての音楽愛好家が面倒なインストールや複雑な設定なしに、素晴らしい曲にアクセスして楽しむことができます。

## 🛠️ 技術情報

- **純粋なHTML、CSS、JavaScript** で構築 – サードパーティ製ライブラリは一切使用していません。
- 内蔵 **ID3v2 パーサー**（v2.3 および v2.4 対応）が、タイトル、アーティスト、アルバム、カバーアートを抽出します。
- **Audio API** と **Media Session API** を使用して、システムレベルのメディア制御を実現。
- すべてのローカライズリソースは埋め込まれており、インターネット接続なしで言語を切り替えられます。

## 📁 サポートされるファイル形式

### オーディオ形式
プレーヤーは、「音楽を追加」ボタンまたはドラッグ＆ドロップで以下のオーディオファイルタイプを開いて再生できます。

- **MP3**（.mp3）
- **WAV**（.wav）
- **FLAC**（.flac）
- **OGG**（.ogg）
- その他一般的な **M4A**、**AAC**、**WMA** など（ブラウザの対応に依存します）。

> プレーヤーはブラウザ内蔵のオーディオエンジンを使用するため、実際の再生機能はブラウザによって若干異なる場合があります。

### メタデータタグ（ID3v2）
MP3ファイル（またはID3v2.3 / v2.4タグを含むファイル）を追加すると、プレーヤーは以下のタグを自動的に読み取ります。

| タグ         | 意味               | 表示される場所       |
|--------------|--------------------|----------------------|
| TIT2         | タイトル           | 曲名                 |
| TPE1         | アーティスト       | アーティスト名       |
| TALB         | アルバム           | アルバム名           |
| TYER / TDRC  | 年 / リリース日    | 年 / 日付            |
| TCOM         | 作曲者             | （単独表示なし、アーティストにフォールバック）|
| TCON         | ジャンル           | ジャンル             |
| TRCK         | トラック番号       | トラック番号         |
| TDAT         | 日付               | 日付                 |
| APIC         | 添付画像           | レコードレーベル上のカバーアート |

ID3タグが見つからない場合、プレーヤーは**ファイル名**を `アーティスト - 曲名` の形式で解析しようとします（最後の ` - ` を区切り文字として使用します）。

### 歌詞形式（LRC）
標準の **LRC** 歌詞ファイル（拡張子 `.lrc`）をインポートできます。パーサーは `[mm:ss.xx]` および `[mm:ss.xxx]` 形式のタイムスタンプをサポートします。

プレーヤーは再生の進行に合わせて現在の歌詞行をリアルタイムでハイライト表示し、任意の行をクリックするとそのタイムスタンプにジャンプできます。

LRCファイルのインポート方法：プレーヤーの **「歌詞」** ボタンをクリックし、**「歌詞をインポート」** を選択するか、対応する曲を再生中に `.lrc` ファイルをページにドラッグ＆ドロップします。
