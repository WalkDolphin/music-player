# 🎵 音樂播放器

> 一個純 HTML 的音樂播放器，帶有復古黑膠唱片介面，支援 150 多種全球語言及近 50 種地方方言。**在同一個 HTML 頁面中，享受全球方言的慢旋律。**

**註：本專案在建立過程中有人工智慧參與。**

---

## ✨ 功能特點

- 🎨 **沉浸式黑膠設計** – 旋轉唱片、唱臂動畫，復古視覺體驗。
- 🌍 **150+ 種語言 & 50 種方言** – 包含吳語、粵語、客家話、贛語、湘語、晉語等。
- 📂 **拖放支援** – 直接拖曳音訊檔案（MP3/WAV/FLAC/OGG）或點擊「新增音樂」。
- 📝 **LRC 歌詞同步** – 匯入 `.lrc` 檔案，即時高亮滾動，並支援全螢幕歌詞模式。
- 🎛️ **完整播放控制** – 播放/暫停、上一首/下一首、隨機播放、循環模式（不循環/全部循環/單曲循環）以及音量調整。
- 📱 **響應式適配** – 在桌面、平板和手機上均能流暢顯示。
- 🧩 **無外部依賴** – 完全在瀏覽器中執行，無需安裝。

---

## 🚀 快速開始

1. 下載 `TheMusicPlayer.html` 檔案。
2. 使用任意現代瀏覽器（Chrome、Edge、Firefox 等）**雙擊開啟**。
3. 點擊 **「新增音樂」** 或將音訊檔案拖放到視窗中。
4. 點擊 **「歌詞」** 按鈕，可為目前歌曲匯入 `.lrc` 歌詞檔案。
5. 點擊 **齒輪圖示（設定）** 可切換字型和介面語言。

---

### 詳情

這款音樂播放器旨在幫助使用者遠離喧囂，伴隨著優美的旋律，擁抱舒緩的慢生活。它提供了全面的本地化支援，覆蓋 150 種全球語言和近 50 種獨特的地方方言。我們首次將沉浸式的復古黑膠唱片視覺設計融入播放介面，帶來更具質感和懷舊感的聆聽體驗——歡迎下載並親自嘗試。

> **設計初衷：** 我的核心靈感源自一個簡單的願望——以輕鬆、不複雜的方式欣賞音樂。我選擇純 HTML 技術建構此播放器，因為它能在所有個人電腦上實現通用相容。這樣，每位音樂愛好者都可以無需繁瑣的安裝或複雜的設定，直接存取並享受美妙的旋律。

---

## 🛠️ 技術說明

- 使用**純 HTML、CSS 和 JavaScript** 建構，無任何第三方函式庫。
- 內建 **ID3v2 解析器**（支援 v2.3 和 v2.4），可提取標題、藝術家、專輯及封面圖。
- 使用 **Audio API** 和 **Media Session API** 實現系統級媒體控制。
- 所有本地化資源均已內嵌，無需連線即可切換語言。

---

## 📁 支援的檔案格式

### 音訊格式
播放器支援透過「新增音樂」按鈕或拖放的方式開啟並播放以下音訊檔案類型：

- **MP3**（.mp3）
- **WAV**（.wav）
- **FLAC**（.flac）
- **OGG**（.ogg）
- 以及常見的 **M4A**、**AAC**、**WMA** 等格式（具體支援情況取決於瀏覽器）。

> 播放器使用瀏覽器內建的音訊引擎，因此實際播放能力可能因瀏覽器而異。

### 中繼資料標籤（ID3v2）
當您新增 MP3 檔案（或包含 ID3v2.3 / v2.4 標籤的其他檔案）時，播放器會自動讀取以下標籤：

| 標籤         | 含義             | 顯示位置             |
|--------------|------------------|----------------------|
| TIT2         | 標題             | 歌曲名稱             |
| TPE1         | 藝術家           | 藝術家名稱           |
| TALB         | 專輯             | 專輯名稱             |
| TYER / TDRC  | 年份 / 發佈日期  | 年份 / 日期          |
| TCOM         | 作曲者           |（無單獨顯示，回退為藝術家）|
| TCON         | 流派             | 流派                 |
| TRCK         | 音軌號           | 音軌號               |
| TDAT         | 日期             | 日期                 |
| APIC         | 封面圖片         | 黑膠唱片標籤上的封面  |

如果未找到 ID3 標籤，播放器會嘗試從**檔案名稱**中解析，格式為 `藝術家 - 歌曲名`（以最後出現的 ` - ` 作為分隔符）。

### 歌詞格式（LRC）
您可以匯入標準的 **LRC** 歌詞檔案（副檔名為 `.lrc`）。解析器支援 `[mm:ss.xx]` 和 `[mm:ss.xxx]` 格式的時間戳記。

播放器會根據播放進度即時高亮目前歌詞行，您也可以點選任意一行直接跳轉到對應時間點。

匯入 LRC 檔案的方式：點擊播放器中的 **「歌詞」** 按鈕，然後選擇 **「匯入歌詞」**；或者在播放對應歌曲時，直接將 `.lrc` 檔案拖放到頁面中。
