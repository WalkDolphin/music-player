# 🎵 뮤직 플레이어

> 빈티지 레코드판 인터페이스를 갖춘 순수 HTML 기반 뮤직 플레이어로, 150개 이상의 글로벌 언어와 약 50개의 지역 방언을 지원합니다. **단 하나의 HTML 페이지에서 전 세계 방언의 느린 멜로디를 즐겨보세요.**

**참고: 이 프로젝트는 인공지능의 도움을 받아 제작되었습니다.**

---

## ✨ 주요 기능

- 🎨 **몰입형 레코드 디자인** – 회전하는 디스크, 톤암 애니메이션, 레트로 비주얼 경험.
- 🌍 **150개 이상의 언어 및 50개 방언** – 우어, 광둥어, 하카어, 간어, 샹어, 진어 등을 포함.
- 📂 **드래그 앤 드롭 지원** – 오디오 파일(MP3/WAV/FLAC/OGG)을 드래그하거나 "음악 추가"를 클릭하세요.
- 📝 **LRC 가사 동기화** – `.lrc` 파일을 가져와 실시간 하이라이트 스크롤 및 전체 화면 가사 모드를 지원합니다.
- 🎛️ **완전한 재생 제어** – 재생/일시정지, 이전/다음, 셔플, 반복 모드(없음/전체/한 곡) 및 볼륨 조절.
- 📱 **반응형** – 데스크톱, 태블릿, 모바일에서 부드럽게 작동합니다.
- 🧩 **외부 의존성 없음** – 브라우저에서 완전히 실행되며 설치가 필요하지 않습니다.

---

## 🚀 빠른 시작

1. `TheMusicPlayer.html` 파일을 다운로드하세요.
2. 최신 브라우저(Chrome, Edge, Firefox 등)에서 **더블 클릭**으로 여세요.
3. **"음악 추가"** 를 클릭하거나 오디오 파일을 창으로 드래그 앤 드롭하세요.
4. **"가사"** 버튼을 클릭하면 현재 곡에 `.lrc` 가사 파일을 가져올 수 있습니다.
5. **톱니바퀴 아이콘(설정)** 을 클릭하면 글꼴과 인터페이스 언어를 전환할 수 있습니다.

### 상세 설명

이 뮤직 플레이어는 사용자가 번잡함에서 벗어나 아름다운 멜로디와 함께 편안한 슬로우 라이프를 누릴 수 있도록 제작되었습니다. 150개 글로벌 언어와 약 50개의 독특한 지역 방언을 포괄하는 완전한 현지화 지원을 제공합니다. 처음으로 빈티지 레코드판 비주얼 디자인을 재생 인터페이스에 통합하여 더욱 질감 있고 향수 어린 청취 경험을 선사합니다. 지금 다운로드하여 직접 체험해 보세요.

> **디자인 의도:** 제 핵심 영감은 편안하고 간단한 방식으로 음악을 감상하고자 하는 단순한 바람에서 비롯되었습니다. 순수 HTML 기술로 이 플레이어를 구축한 이유는 모든 개인용 컴퓨터에서 보편적인 호환성을 제공하기 때문입니다. 이를 통해 모든 음악 애호가가 번거로운 설치나 복잡한 설정 없이도 멋진 곡에 접근하고 즐길 수 있습니다.

## 🛠️ 기술 노트

- **순수 HTML, CSS, JavaScript**로 구축 – 타사 라이브러리 없음.
- 내장 **ID3v2 파서**(v2.3 및 v2.4 지원)가 제목, 아티스트, 앨범, 커버 아트를 추출합니다.
- **Audio API** 및 **Media Session API**를 사용하여 시스템 수준의 미디어 제어를 구현합니다.
- 모든 현지화 리소스가 내장되어 있어 인터넷 연결 없이도 언어를 전환할 수 있습니다.

## 📁 지원하는 파일 형식

### 오디오 형식
플레이어는 "음악 추가" 버튼이나 드래그 앤 드롭을 통해 다음 오디오 파일 형식을 열고 재생할 수 있습니다.

- **MP3** (.mp3)
- **WAV** (.wav)
- **FLAC** (.flac)
- **OGG** (.ogg)
- 그리고 일반적인 **M4A**, **AAC**, **WMA** 등 (브라우저 지원 여부에 따라 다름).

> 플레이어는 브라우저 내장 오디오 엔진을 사용하므로 실제 재생 기능은 브라우저에 따라 약간 다를 수 있습니다.

### 메타데이터 태그 (ID3v2)
MP3 파일(또는 ID3v2.3 / v2.4 태그를 포함하는 파일)을 추가하면 플레이어가 다음 태그를 자동으로 읽습니다.

| 태그         | 의미                     | 표시 위치                 |
|--------------|--------------------------|---------------------------|
| TIT2         | 제목                     | 곡명                      |
| TPE1         | 아티스트                 | 아티스트명                |
| TALB         | 앨범                     | 앨범명                    |
| TYER / TDRC  | 연도 / 출시일            | 연도 / 날짜               |
| TCOM         | 작곡가                   | (별도 표시 없음, 아티스트로 대체) |
| TCON         | 장르                     | 장르                      |
| TRCK         | 트랙 번호                | 트랙 번호                 |
| TDAT         | 날짜                     | 날짜                      |
| APIC         | 첨부 이미지              | 레코드 레이블의 커버 아트 |

ID3 태그가 없으면 플레이어는 **파일 이름**을 `아티스트 - 곡명` 형식으로 파싱하려고 시도합니다(마지막 ` - `를 구분자로 사용).

### 가사 형식 (LRC)
표준 **LRC** 가사 파일(`.lrc` 확장자)을 가져올 수 있습니다. 파서는 `[mm:ss.xx]` 및 `[mm:ss.xxx]` 형식의 타임스탬프를 지원합니다.

플레이어는 재생 진행에 따라 현재 가사 줄을 실시간으로 강조 표시하며, 아무 줄이나 클릭하면 해당 타임스탬프로 점프할 수 있습니다.

LRC 파일 가져오기: 플레이어의 **"가사"** 버튼을 클릭하고 **"가사 가져오기"** 를 선택하거나, 해당 곡을 재생하는 동안 `.lrc` 파일을 페이지로 드래그 앤 드롭하세요.
