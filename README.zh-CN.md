# 🎵 音乐播放器

> 一个纯 HTML 的音乐播放器，带有复古黑胶唱片界面，支持 150 多种全球语言及近 50 种地方方言。**在同一个 HTML 页面中，享受全球方言的慢旋律。**

**注：本项目在创建过程中有人工智能参与。**

---

## ✨ 功能特点

- 🎨 **沉浸式黑胶设计** – 旋转唱片、唱臂动画，复古视觉体验。
- 🌍 **150+ 种语言 & 50 种方言** – 包含吴语、粤语、客家话、赣语、湘语、晋语等。
- 📂 **拖放支持** – 直接拖拽音频文件（MP3/WAV/FLAC/OGG）或点击“添加音乐”。
- 📝 **LRC 歌词同步** – 导入 `.lrc` 文件，实时高亮滚动，并支持全屏歌词模式。
- 🎛️ **完整播放控制** – 播放/暂停、上一首/下一首、随机播放、循环模式（不循环/全部循环/单曲循环）以及音量调节。
- 📱 **响应式适配** – 在桌面、平板和手机上均能流畅显示。
- 🧩 **无外部依赖** – 完全在浏览器中运行，无需安装。

---

## 🚀 快速开始

1. 下载 `TheMusicPlayer.html` 文件。
2. 使用任意现代浏览器（Chrome、Edge、Firefox 等）**双击打开**。
3. 点击 **“添加音乐”** 或将音频文件拖放到窗口中。
4. 点击 **“歌词”** 按钮，可为当前歌曲导入 `.lrc` 歌词文件。
5. 点击 **齿轮图标（设置）** 可切换字体和界面语言。

## 🛠️ 技术说明

- 使用**纯 HTML、CSS 和 JavaScript** 构建，无任何第三方库。
- 内置 **ID3v2 解析器**（支持 v2.3 和 v2.4），可提取标题、艺术家、专辑及封面图。
- 使用 **Audio API** 和 **Media Session API** 实现系统级媒体控制。
- 所有本地化资源均已内嵌，无需联网即可切换语言。

## 📁 支持的文件格式

### 音频格式
播放器支持通过“添加音乐”按钮或拖放的方式打开并播放以下音频文件类型：

- **MP3**（.mp3）
- **WAV**（.wav）
- **FLAC**（.flac）
- **OGG**（.ogg）
- 以及常见的 **M4A**、**AAC**、**WMA** 等格式（具体支持情况取决于浏览器）。

> 播放器使用浏览器内置的音频引擎，因此实际播放能力可能因浏览器而异。

### 元数据标签（ID3v2）
当您添加 MP3 文件（或包含 ID3v2.3 / v2.4 标签的其他文件）时，播放器会自动读取以下标签：

| 标签         | 含义             | 显示位置             |
|--------------|------------------|----------------------|
| TIT2         | 标题             | 歌曲名称             |
| TPE1         | 艺术家           | 艺术家名称           |
| TALB         | 专辑             | 专辑名称             |
| TYER / TDRC  | 年份 / 发布日期  | 年份 / 日期          |
| TCOM         | 作曲者           |（无单独显示，回退为艺术家）|
| TCON         | 流派             | 流派                 |
| TRCK         | 音轨号           | 音轨号               |
| TDAT         | 日期             | 日期                 |
| APIC         | 封面图片         | 黑胶唱片标签上的封面  |

如果未找到 ID3 标签，播放器会尝试从**文件名**中解析，格式为 `艺术家 - 歌曲名`（以最后出现的 ` - ` 作为分隔符）。

### 歌词格式（LRC）
您可以导入标准的 **LRC** 歌词文件（扩展名为 `.lrc`）。解析器支持 `[mm:ss.xx]` 和 `[mm:ss.xxx]` 格式的时间戳。

播放器会根据播放进度实时高亮当前歌词行，您也可以点击任意一行直接跳转到对应时间点。

导入 LRC 文件的方式：点击播放器中的 **“歌词”** 按钮，然后选择 **“导入歌词”**；或者在播放对应歌曲时，直接将 `.lrc` 文件拖放到页面中。
