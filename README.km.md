# 🎵 កម្មវិធីចាក់តន្ត្រី

> កម្មវិធីចាក់តន្ត្រី HTML សុទ្ធ ជាមួយចំណុចប្រទាក់ថាសវីនីលបែបវីនីតាជែ (vintage) ដែលគាំទ្រភាសាសកលជាង ១៥០ និងគ្រាមភាសាក្នុងតំបន់ជិត ៥០។ **នៅក្នុងទំព័រ HTML តែមួយ សូមរីករាយជាមួយបទភ្លេងយឺតៗ ជាមួយគ្រាមភាសាមកពីជុំវិញពិភពលោក។**

**ចំណាំ៖ គម្រោងនេះត្រូវបានបង្កើតឡើងដោយមានការចូលរួមពីបញ្ញាសិប្បនិម្មិត (AI)។**

## ✨ លក្ខណៈពិសេស

- 🎨 **ការរចនាថាសវីនីលបែបអន្លង់អន្លូន** – ថាសបង្វិល ចលនាដៃចាក់ បទពិសោធន៍រូបភាពបែប Retro។
- 🌍 **ភាសាជាង ១៥០ និងគ្រាមភាសា ៥០** – រួមមាន វូ (Wu), កន្តាំង, ហាក់កា (Hakka), កាន (Gan), ស៊ាង (Xiang), ជិន (Jin) និងផ្សេងទៀត។
- 📂 **គាំទ្រការអូសនិងទម្លាក់** – អូសឯកសារអូឌីយ៉ូ (MP3/WAV/FLAC/OGG) ឬចុច "បន្ថែមតន្ត្រី"។
- 📝 **ការធ្វើសមកាលកម្មទំនុកច្រៀង LRC** – នាំចូលឯកសារ `.lrc` ជាមួយការរំលេចអត្ថបទអំឡុងពេលរំកិល និងរបៀបទំព័រពេញ។
- 🎛️ **ការគ្រប់គ្រងការចាក់សារឡើងវិញពេញលេញ** – ចាក់/ផ្អាក, មុន/បន្ទាប់, សាច់ច្រឡំ (Shuffle), របៀបចាក់ឡើងវិញ (គ្មាន/ទាំងអស់/មួយបទ) និងការលៃតម្រូវកម្រិតសំឡេង។
- 📱 **ឆ្លើយតបនឹងអេក្រង់** – ដំណើរការយ៉ាងរលូនលើកុំព្យូទ័រលើតុ ថេប្លេត និងទូរស័ព្ទដៃ។
- 🧩 **មិនពឹងផ្អែកលើកម្មវិធីខាងក្រៅ** – ដំណើរការទាំងស្រុងនៅក្នុងកម្មវិធីរុករក មិនត្រូវការដំឡើងទេ។

### សេចក្តីលម្អិត

កម្មវិធីចាក់តន្ត្រីនេះត្រូវបានបង្កើតឡើងដើម្បីជួយអ្នកប្រើប្រាស់ឱ្យឃ្លាតឆ្ងាយពីភាពមមាញឹក និងទទួលយកជីវិតយឺតៗ និងស្ងប់ស្ងាត់ អមដោយបទភ្លេងដ៏ពិរោះ។ វាផ្តល់នូវការគាំទ្រការធ្វើមូលដ្ឋានីយកម្មយ៉ាងទូលំទូលាយគ្របដណ្តប់ ១៥០ ភាសាសកល និងគ្រាមភាសាក្នុងតំបន់ប្លែកៗជិត ៥០។ ជាលើកដំបូង យើងបានបញ្ចូលការរចនារូបភាពថាសវីនីលបែបវីនីតាជែ (vintage) ដ៏អន្លង់អន្លូនទៅក្នុងចំណុចប្រទាក់ចាក់សារ ដែលផ្តល់នូវបទពិសោធន៍ស្តាប់ដែលមានអារម្មណ៍ និងនឹករលឹកកាន់តែច្រើន។ សូមទាញយក និងសាកល្បងដោយខ្លួនឯង។

> **ចេតនានៃការរចនា៖** ការបំផុសគំនិតចម្បងរបស់ខ្ញុំកើតចេញពីបំណងប្រាថ្នាដ៏សាមញ្ញមួយ គឺដើម្បីឱ្យតម្លៃចំពោះតន្ត្រីក្នុងវិធីសម្រាកកាយ និងមិនស្មុគស្មាញ។ ខ្ញុំបានជ្រើសរើសបង្កើតកម្មវិធីចាក់នេះដោយប្រើបច្ចេកវិទ្យា HTML សុទ្ធ ព្រោះវាផ្តល់នូវភាពឆបគ្នាជាសកលនៅលើកុំព្យូទ័រផ្ទាល់ខ្លួនទាំងអស់។ តាមវិធីនេះ អ្នកស្រឡាញ់តន្ត្រីគ្រប់រូបអាចចូលប្រើ និងរីករាយជាមួយបទភ្លេងដ៏អស្ចារ្យ ដោយមិនចាំបាច់ដំឡើងដ៏ស្មុគស្មាញ ឬការកំណត់រចនាសម្ព័ន្ធដ៏លំបាកនោះទេ។

## 🛠️ កំណត់ចំណាំបច្ចេកទេស

- បង្កើតឡើងជាមួយ **HTML, CSS និង JavaScript សុទ្ធ** – ដោយគ្មានបណ្ណាល័យភាគីទីបី។
- **កម្មវិធីញែក ID3v2** ដែលភ្ជាប់មកជាមួយ (គាំទ្រ v2.3 និង v2.4) ទាញយកចំណងជើង សិល្បករ អាល់ប៊ុម និងគម្រប។
- ប្រើប្រាស់ **Audio API** និង **Media Session API** សម្រាប់ការគ្រប់គ្រងប្រព័ន្ធផ្សព្វផ្សាយកម្រិតប្រព័ន្ធ។
- ធនធានធ្វើមូលដ្ឋានីយកម្មទាំងអស់ត្រូវបានបង្កប់នៅក្នុង – មិនត្រូវការភ្ជាប់អ៊ីនធឺណិតដើម្បីប្តូរភាសាទេ។

## 📁 ទ្រង់ទ្រាយឯកសារដែលគាំទ្រ

### ទ្រង់ទ្រាយអូឌីយ៉ូ
កម្មវិធីចាក់អាចបើក និងចាក់ប្រភេទឯកសារអូឌីយ៉ូដូចខាងក្រោម (តាមរយៈប៊ូតុង "បន្ថែមតន្ត្រី" ឬអូសនិងទម្លាក់)៖

- **MP3** (.mp3)
- **WAV** (.wav)
- **FLAC** (.flac)
- **OGG** (.ogg)
- ព្រមទាំងទ្រង់ទ្រាយទូទៅដូចជា **M4A**, **AAC** និង **WMA** (អាស្រ័យលើការគាំទ្ររបស់កម្មវិធីរុករក)។

> កម្មវិធីចាក់ប្រើម៉ាស៊ីនអូឌីយ៉ូដែលភ្ជាប់មកជាមួយរបស់កម្មវិធីរុករក ដូច្នេះសមត្ថភាពចាក់សារជាក់ស្តែងអាចខុសគ្នាបន្តិចបន្តួចរវាងកម្មវិធីរុករកនីមួយៗ។

### ទ្រង់ទ្រាយទំនុកច្រៀង (LRC)
អ្នកអាចនាំចូលឯកសារទំនុកច្រៀងតាមស្តង់ដារ **LRC** (ដែលមានផ្នែកបន្ថែម `.lrc`)។ កម្មវិធីញែកគាំទ្រសញ្ញាពេលវេលាតាមទ្រង់ទ្រាយដូចខាងក្រោម៖

- `[mm:ss.xx]` – នាទី វិនាទី និងភាគរយវិនាទី
