# 🎵 โปรแกรมเล่นเพลง

> โปรแกรมเล่นเพลงในรูปแบบ HTML ล้วน พร้อมอินเทอร์เฟซแผ่นเสียงไวนิลวินเทจ รองรับมากกว่า 150 ภาษาและเกือบ 50 ภาษาถิ่นระดับภูมิภาค **ในหน้า HTML เดียว เพลิดเพลินกับท่วงทำนองช้า ๆ พร้อมภาษาถิ่นจากทั่วโลก**

**หมายเหตุ: โปรเจกต์นี้มีการมีส่วนร่วมจากปัญญาประดิษฐ์ในการสร้างสรรค์**

---

## ✨ คุณสมบัติ

- 🎨 **ดีไซน์แผ่นเสียงที่สมจริง** – จานหมุน แอนิเมชันแขนเข็ม ประสบการณ์ภาพย้อนยุค
- 🌍 **150+ ภาษา และ 50 ภาษาถิ่น** – รวมถึงอู๋, กวางตุ้ง, แคะ, กั้น, เซียง, จิ้น และอื่น ๆ
- 📂 **รองรับการลากและวาง** – ลากไฟล์เสียง (MP3/WAV/FLAC/OGG) หรือคลิก "เพิ่มเพลง"
- 📝 **ซิงโครไนซ์เนื้อเพลง LRC** – นำเข้าไฟล์ `.lrc` พร้อมไฮไลต์เลื่อนและโหมดเนื้อเพลงเต็มหน้าจอ
- 🎛️ **ควบคุมการเล่นอย่างครบครัน** – เล่น/หยุดชั่วคราว, ก่อนหน้า/ถัดไป, สุ่ม, โหมดเล่นซ้ำ (ไม่ซ้ำ/ทั้งหมด/หนึ่งเพลง) และปรับระดับเสียง
- 📱 **ปรับตามอุปกรณ์** – ทำงานได้อย่างราบรื่นบนเดสก์ท็อป แท็บเล็ต และมือถือ
- 🧩 **ไม่พึ่งพาภายนอก** – ทำงานในเบราว์เซอร์ทั้งหมด ไม่ต้องติดตั้ง

---

## 🚀 เริ่มต้นใช้งาน

1. ดาวน์โหลดไฟล์ `TheMusicPlayer.html`
2. **ดับเบิลคลิก** เพื่อเปิดในเบราว์เซอร์สมัยใหม่ (Chrome, Edge, Firefox, ฯลฯ)
3. คลิก **"เพิ่มเพลง"** หรือลากและวางไฟล์เสียงลงในหน้าต่าง
4. คลิกปุ่ม **"เนื้อเพลง"** เพื่อนำเข้าไฟล์ `.lrc` สำหรับเพลงปัจจุบัน
5. คลิก **ไอคอนเฟือง (การตั้งค่า)** เพื่อเปลี่ยนแบบอักษรและภาษาของอินเทอร์เฟซ

### รายละเอียด

โปรแกรมเล่นเพลงนี้ถูกสร้างขึ้นเพื่อช่วยให้ผู้ใช้หลีกหนีจากความวุ่นวายและโอบรับชีวิตที่ช้าและผ่อนคลายพร้อมกับท่วงทำนองที่สวยงาม มาพร้อมกับการรองรับการแปลภาษาที่ครอบคลุมกว่า 150 ภาษาและเกือบ 50 ภาษาถิ่นระดับภูมิภาค เป็นครั้งแรกที่เราได้ผสานรวมดีไซน์แผ่นเสียงไวนิลวินเทจที่สมจริงเข้ากับอินเทอร์เฟซการเล่น มอบประสบการณ์การฟังที่สัมผัสได้และหวนคิดถึงอดีตมากขึ้น—ดาวน์โหลดและลองใช้งานด้วยตัวคุณเองได้เลย

> **เจตนารมณ์ในการออกแบบ:** แรงบันดาลใจหลักของฉันมาจากความปรารถนาง่าย ๆ ในการชื่นชมดนตรีอย่างผ่อนคลายและไม่ซับซ้อน ฉันเลือกสร้างโปรแกรมเล่นนี้ด้วยเทคโนโลยี HTML ล้วน เพราะมันมอบความเข้ากันได้อย่างเป็นสากลบนคอมพิวเตอร์ส่วนบุคคลทุกเครื่อง ด้วยวิธีนี้ ผู้รักดนตรีทุกคนสามารถเข้าถึงและเพลิดเพลินกับท่วงทำนองที่ยอดเยี่ยมได้โดยไม่ต้องติดตั้งยุ่งยากหรือตั้งค่าที่ซับซ้อน

## 🛠️ หมายเหตุทางเทคนิค

- สร้างด้วย **HTML, CSS และ JavaScript ล้วน** – ไม่มีไลบรารีของบุคคลที่สาม
- มี **ตัวแยกวิเคราะห์ ID3v2 ในตัว** (รองรับ v2.3 และ v2.4) ที่ดึงข้อมูลชื่อเพลง ศิลปิน อัลบั้ม และภาพปก
- ใช้ **Audio API** และ **Media Session API** เพื่อควบคุมสื่อในระดับระบบ
- ทรัพยากรการแปลภาษาทั้งหมดถูกฝังไว้ในตัว – ไม่ต้องเชื่อมต่ออินเทอร์เน็ตเพื่อเปลี่ยนภาษา

## 📁 รูปแบบไฟล์ที่รองรับ

### รูปแบบเสียง
โปรแกรมเล่นสามารถเปิดและเล่นไฟล์เสียงประเภทต่อไปนี้ (ผ่านปุ่ม "เพิ่มเพลง" หรือการลากและวาง):

- **MP3** (.mp3)
- **WAV** (.wav)
- **FLAC** (.flac)
- **OGG** (.ogg)
- รวมถึงรูปแบบทั่วไปเช่น **M4A**, **AAC**, และ **WMA** (ขึ้นอยู่กับการรองรับของเบราว์เซอร์)

> โปรแกรมเล่นใช้เอ็นจินเสียงในตัวของเบราว์เซอร์ ดังนั้นความสามารถในการเล่นจริงอาจแตกต่างกันเล็กน้อยในแต่ละเบราว์เซอร์

### แท็กข้อมูลเมตา (ID3v2)
เมื่อคุณเพิ่มไฟล์ MP3 (หรือไฟล์ใด ๆ ที่มีแท็ก ID3v2.3 / v2.4) โปรแกรมเล่นจะอ่านแท็กต่อไปนี้โดยอัตโนมัติ:

| แท็ก         | ความหมาย               | แสดงเป็น               |
|--------------|------------------------|------------------------|
| TIT2         | ชื่อเพลง               | ชื่อเพลง               |
| TPE1         | ศิลปิน                 | ชื่อศิลปิน             |
| TALB         | อัลบั้ม                | ชื่ออัลบั้ม            |
| TYER / TDRC  | ปี / วันที่วางจำหน่าย  | ปี / วันที่             |
| TCOM         | ผู้แต่งเพลง            | (ไม่แสดงแยก ใช้ศิลปินแทน) |
| TCON         | แนวเพลง               | แนวเพลง                |
| TRCK         | หมายเลขแทร็ก           | หมายเลขแทร็ก           |
| TDAT         | วันที่                 | วันที่                  |
| APIC         | รูปภาพที่แนบมา         | ภาพปกบนฉลากแผ่นเสียง   |

หากไม่พบแท็ก ID3 โปรแกรมเล่นจะพยายามแยกวิเคราะห์จาก**ชื่อไฟล์**ในรูปแบบ `ศิลปิน - ชื่อเพลง` (โดยใช้ ` - ` ตัวสุดท้ายเป็นตัวคั่น)

### รูปแบบเนื้อเพลง (LRC)
คุณสามารถนำเข้าไฟล์เนื้อเพลงแบบซิงโครไนซ์มาตรฐาน **LRC** (ที่มีนามสกุล `.lrc`) ตัวแยกวิเคราะห์รองรับรูปแบบเวลา:

- `[mm:ss.xx]` – นาที วินาที และร้อยวินาที
- `[mm:ss.xxx]` – นาที วินาที และพันวินาที

โปรแกรมเล่นจะไฮไลต์บรรทัดเนื้อเพลงปัจจุบันแบบเรียลไทม์ตามความคืบหน้าของการเล่น และคุณสามารถคลิกที่บรรทัดใดก็ได้เพื่อข้ามไปยังเวลานั้น

วิธีนำเข้าไฟล์ LRC: คลิกปุ่ม **"เนื้อเพลง"** ในโปรแกรมเล่น แล้วเลือก **"นำเข้าเนื้อเพลง"** หรือในขณะที่เพลงที่เกี่ยวข้องกำลังเล่น ให้ลากและวางไฟล์ `.lrc` ลงบนหน้าเพจโดยตรง
