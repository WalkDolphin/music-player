# 🎵 Музичний програвач

> Музичний програвач на чистому HTML з інтерфейсом вінтажної вінілової платівки, який підтримує понад 150 мов світу та майже 50 регіональних діалектів. **В одній HTML-сторінці насолоджуйтесь повільними мелодіями з діалектами з усього світу.**

**Примітка: цей проєкт було створено за участі штучного інтелекту.**

---

## ✨ Можливості

- 🎨 **Імерсивний вініловий дизайн** – обертовий диск, анімація тонарма, ретро-візуальний досвід.
- 🌍 **Понад 150 мов і 50 діалектів** – включаючи ву, кантонську, хакка, гань, сян, цзінь та інші.
- 📂 **Підтримка перетягування** – перетягніть аудіофайли (MP3/WAV/FLAC/OGG) або натисніть "Додати музику".
- 📝 **Синхронізація текстів LRC** – імпортуйте файли `.lrc` з підсвічуванням під час прокручування та повноекранним режимом текстів.
- 🎛️ **Повне керування відтворенням** – відтворення/пауза, попередній/наступний, перемішування, режими повтору (немає/всі/одна) та регулювання гучності.
- 📱 **Адаптивність** – працює бездоганно на настільних комп'ютерах, планшетах і мобільних пристроях.
- 🧩 **Без зовнішніх залежностей** – працює повністю в браузері, не потребує встановлення.

---

## 🚀 Швидкий старт

1. Завантажте файл `TheMusicPlayer.html`.
2. Відкрийте його **подвійним клацанням** у будь-якому сучасному браузері (Chrome, Edge, Firefox тощо).
3. Натисніть **"Додати музику"** або перетягніть аудіофайли у вікно.
4. Натисніть кнопку **"Тексти"**, щоб імпортувати файл `.lrc` для поточної пісні.
5. Натисніть **значок шестірні (налаштування)**, щоб змінити шрифт і мову інтерфейсу.

## 🛠️ Технічні примітки

- Створено на **чистому HTML, CSS та JavaScript** – без сторонніх бібліотек.
- Вбудований **аналізатор ID3v2** (підтримує v2.3 і v2.4) витягує назву, виконавця, альбом і обкладинку.
- Використовує **Audio API** та **Media Session API** для керування медіа на системному рівні.
- Усі ресурси локалізації вбудовані – для зміни мови не потрібне підключення до Інтернету.

## 📁 Підтримувані формати файлів

### Аудіоформати
Програвач може відкривати та відтворювати такі типи аудіофайлів (через кнопку "Додати музику" або перетягуванням):

- **MP3** (.mp3)
- **WAV** (.wav)
- **FLAC** (.flac)
- **OGG** (.ogg)
- А також поширені формати, як-от **M4A**, **AAC** та **WMA** (залежно від підтримки браузером).

> Програвач використовує вбудований аудіодвигун браузера, тому фактична здатність відтворення може дещо відрізнятися між браузерами.

### Теги метаданих (ID3v2)
Коли ви додаєте MP3-файл (або будь-який файл, що містить теги ID3v2.3 / v2.4), програвач автоматично зчитує такі теги:

| Тег         | Значення                     | Відображається як         |
|-------------|------------------------------|---------------------------|
| TIT2        | Назва                        | Назва пісні               |
| TPE1        | Виконавець                   | Ім'я виконавця            |
| TALB        | Альбом                       | Назва альбому             |
| TYER / TDRC | Рік / Дата випуску           | Рік / Дата                |
| TCOM        | Композитор                   | (не відображається окремо, використовується виконавець) |
| TCON        | Жанр                         | Жанр                      |
| TRCK        | Номер треку                  | Номер треку               |
| TDAT        | Дата                         | Дата                      |
| APIC        | Прикріплене зображення       | Обкладинка на етикетці платівки |

Якщо теги ID3 не знайдено, програвач спробує проаналізувати **ім'я файлу** у форматі `Виконавець - Назва` (останній ` - ` використовується як роздільник).

### Формат текстів пісень (LRC)
Ви можете імпортувати стандартні файли синхронізованих текстів **LRC** (з розширенням `.lrc`). Аналізатор підтримує позначки часу в таких форматах:

- `[mm:ss.xx]` – хвилини, секунди та соті секунди
- `[mm:ss.xxx]` – хвилини, секунди та тисячні секунди

Програвач виділяє поточний рядок тексту в реальному часі під час відтворення, і ви можете клацнути будь-який рядок, щоб перейти до цієї позначки часу.

Як імпортувати файл LRC: натисніть кнопку **"Тексти"** у програвачі та виберіть **"Імпортувати тексти"**; або під час відтворення відповідної пісні перетягніть файл `.lrc` безпосередньо на сторінку.
